# Proyecto Final – Sitio Web WordPress con Pasarela de Pago

Este proyecto corresponde al trabajo final del componente **Portafolio II**, cuyo objetivo fue desarrollar un sitio web de comercio electrónico utilizando **WordPress y WooCommerce**, integrando una **pasarela de pago funcional** y ejecutándolo en un entorno local mediante **XAMPP**.

##  Archivos del proyecto

- `installer.php` → Instalador generado con el plugin Duplicator.
- Archivo `.daf` → Respaldo completo del sitio web (alojado externamente debido a su tamaño).

 **Descarga del archivo .daf:**  
https://drive.google.com/file/d/1aUd7c9X8IQ5BV7vXdPYTLUgG1aOym1q0/view?usp=sharing

##  Requisitos

- XAMPP (Apache y MySQL)
- Navegador web actualizado
- Sistema operativo Windows

## Instalación y ejecución en entorno local

1. Descargar `installer.php` desde este repositorio.
2. Descargar el archivo `.daf` desde el enlace indicado.
3. Crear una carpeta dentro de:
   `C:\xampp\htdocs\`
   Ejemplo: `montanostudio`
4. Copiar ambos archivos (`installer.php` y `.daf`) dentro de la carpeta creada.
5. Iniciar **Apache** y **MySQL** desde el panel de control de XAMPP.
6. Abrir el navegador y acceder a:
   `http://localhost/montanostudio/installer.php`
7. Crear una base de datos en **phpMyAdmin** (por ejemplo: `wordpress`).
8. Seguir los pasos del instalador de **Duplicator**.
9. Finalizar la instalación.

##  Resultado esperado

Una vez finalizado el proceso, el sitio web quedará completamente funcional en entorno local, incluyendo:
- Páginas del sitio web.
- Productos.
- Carrito de compras.
- Pasarela de pago.
- Panel administrativo de WordPress.
